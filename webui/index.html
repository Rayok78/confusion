Add Contact

To add a new contact, you need to either create an invite code to send to your contact, or accept an invite code sent to you from a contact:

Create Invite Code

Please enter a name for your contact, and check the invite code to make sure it's to your liking. If you would like a different invite code, click the refresh button.

Give Invite Code to Bob

Please give this code to Bob:

Here are some suggested ways you can give this code to Bob securely:

  • In person
  • Over an existing secure channel
  • Over the phone

Confusion's public key cryptography ensures confidentiality if this invite code is compromised (e.g. if you give it out over a wiretapped phone), but keeping it secret can't hurt and helps better protect your anonymity.

Waiting for Bob to accept invite

Bob will be automatically added to your contacts after the invite is accepted.
